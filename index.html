Clients e Servers

Clients são dispositivos que acessam serviços. Já os Servers são servidores que gerenciam esse serviço enviando informações para o clients, ou seja, os Clients pedem coisas(informações) e os servers fornecem.

Entendendo sobre HTML

HTML

Html(Hypertext Markup Language) é uma linguagem de marcação, ou seja um conjunto de códigou ou tags que montam a estrutura e o layout de um documento digital. Ele não é uma linguagem de programação, pois não cria funcionalidades dinâmicas.

Tags

As Tags são utilizadas para conceituar partes do conteúdo e exibi-los.

Atributos

De forma simples, atributos lembram "adjetivos". Eles "trazem" informações adicionais sobre como deve se tratar um elemento

Listas

As listas são formas de organizar itens, podendo ser de forma ordenada ou não-ordenada. A diferença de uma para outra é que a lista ordenada é formada por números ao seu lado.

Ex.:
  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Já as listas não-ordenadas, ao invés de números ao seu lado, contém "círculos"

Ex.:
  • Primeiro item
  • Segundo item
  • Terceiro item
